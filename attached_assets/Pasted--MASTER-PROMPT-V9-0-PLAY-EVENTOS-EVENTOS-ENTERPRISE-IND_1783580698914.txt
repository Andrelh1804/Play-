# MASTER PROMPT V9.0 — PLAY+EVENTOS EVENTOS ENTERPRISE

# INDUSTRIALIZAÇÃO COMPLETA • ENTERPRISE SAAS • PRODUÇÃO GLOBAL

## MISSÃO

Atue como um Comitê Internacional formado por Distinguished Engineers, Principal Software Architects, Enterprise Architects, Cloud Architects, Especialistas em SaaS Multi-Tenant, Especialistas em Ticketing, ERP, CRM, DevSecOps, SRE, Banco de Dados, UX, QA, Observabilidade, Segurança da Informação, Inteligência Artificial e Eventos.

Sua missão é transformar definitivamente a PLAY+EVENTOS em uma plataforma SaaS Enterprise pronta para comercialização nacional e internacional.

O foco desta fase não é criar telas novas, mas consolidar, integrar, padronizar, automatizar e preparar toda a plataforma para operar em produção com alta disponibilidade, segurança e escalabilidade.

Todo o desenvolvimento deverá preservar a arquitetura existente e utilizar apenas dados reais.

# CRITÉRIOS DE ACEITAÇÃO

A fase será considerada concluída quando:

* Todos os módulos compartilharem dados de forma consistente.
* Não existirem telas, menus ou componentes duplicados.
* Todo CRUD seguir o padrão corporativo.
* Todos os workflows funcionarem de forma automatizada.
* O Ticket Enterprise estiver totalmente integrado ao ERP, CRM e Financeiro.
* A Landing Page refletir automaticamente os eventos publicados.
* O Marketplace permitir gestão completa de fornecedores e patrocinadores.
* Todos os processos críticos possuírem auditoria.
* A plataforma estiver preparada para operação Multi-Tenant.
* A documentação técnica estiver atualizada.
* Não existirem erros de compilação ou funcionalidades simuladas.

# RESULTADO ESPERADO

Ao final desta fase, a PLAY+EVENTOS deverá operar como uma plataforma SaaS Enterprise totalmente integrada, preparada para produção, comercialização e expansão. O sistema deverá oferecer uma experiência unificada para organizadores, participantes, patrocinadores, fornecedores e administradores, com processos automatizados, governança robusta, dados consistentes, alta disponibilidade e arquitetura preparada para evolução contínua.
